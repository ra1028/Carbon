Section

public struct Section

Represents a section of list UI, containing header node, footer node and collection of cell nodes. This works as an intermediary for DifferenceKit.

Example for simple section.

Section(
    id: 0,
    header: Label("Header"),
    footer: Label("Footer"),
    cells: {
        Label("Cell 0")
            .identified(by: \.text)

        Label("Cell 1")
            .identified(by: \.text)

        Label("Cell 2")
            .identified(by: \.text)
    }
)
